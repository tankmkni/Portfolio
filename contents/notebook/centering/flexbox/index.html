メモ：flexboxでのセンター寄せ

いろいろなことが出来そうなflexboxだが、今回は中央寄せだけを調べる
中央寄せしたい要素の親要素に下記を指定 
ParlentElement{
  display:          flex;    /* flexboxの指定 */
  justify-content:  center;  /* 子要素の横方向の配置 */
  align-items:      center;  /* 子要素の縦方向の配置 */
}
test1.html

① 子要素が上下左右中央に配置されている
② pタグが中央配置になったが、サイズが親要素依存から内容依存に変わっている
③ サイズ指定は特に問題ない
④ 要素内のテキストにも flex が適用されている
⑤ pタグにサイズ指定していなくても、内容には flex が適用されている
