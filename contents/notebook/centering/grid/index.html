メモ：gridでのセンター寄せ

gridレイアウトでもいろいろできそうだが、今回は中央寄せだけを調べる
中央寄せしたい要素の親要素に下記を指定 
ParlentElement{
  display:          grid;    /* gridレイアウトの指定 */
  justify-content:  center;  /* 子要素の横方向の配置 */
  align-items:      center;  /* 子要素の縦方向の配置 */
}
test1.html

① 子要素が左右センターに配置されている
② 子要素が上下センターに配置されている
③ 子要素が上下左右センターに配置されている
place-items: center; なら、１行で上下左右の配置を指定できる
test2.html

① 子要素が上下左右センターに配置されている(pタグ内のテキストにも適用)
② しかし、flexboxだと上下にしか適用されていない
③ 何も指定しなければ、左上に配置
