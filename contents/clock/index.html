Clocks

ソースのダウンロード
作品プロフィール
使用言語：
HTML / CSS / JavaScript

概要：
上段左の時計は同級生が<canvas>で作成した時計です。
そのコードを基にあれこれと付け加えたのが他の時計です。
下段の時計の画像の１つも同級生が作ってくれたものを拝借しています。
CSSでの要素の配置に苦労しました。

コスモクロック風（上段右）：
  • 秒針が増えていく
  • 3～6秒ごとにランダムで色を変える
  • 00秒で秒針を全てリセット
  • デジタル時計部をクリックすると、
    　ランダムの色変化パターン(3種)が変わる

    • 秒針を画像で表現（下段）：
  • 秒針代わりに画像を回転
  • クリックすると、針の色がランダムで変わる
  • 同時に、回転のパターンが切り替わる
    　(クォーツ風 ⇔ 機械式風)
  • 画像のsrcを変えるだけでイメージが変えられる
    • 戻る
